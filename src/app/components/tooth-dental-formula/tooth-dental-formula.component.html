Правила заполнения
Указываются реквизиты клиента, осуществляющего перевод средств:
  • Номер счета клиента, с которого списываются средства при переводе;
  • Наименование клиента: полное или краткое, предусмотренное учредительными документами клиента; ФИО- для физ.лиц;
  • Адрес;
  • Cерия и номер паспорта для физ.лиц без открытия счета
(Обязательно для заполнения)
Формат поля
Максимально 5 строк по 35 символов, включая строку с номером счета
  • 1/Наим.клиента или ФИО
  • 2/адрес
  • 3/код страны/город
  • 6/код страны/TXID/ИНН- для юр.лиц
  • 6/код страны/CCPT/серия и номер паспорта для физ. лиц
